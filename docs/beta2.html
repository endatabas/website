Endatabas is an immutable database. It profits from today's confluence of scale-out computing power, limitless disk space, and machine learning.

Contemporary to this watershed moment in tech is one of cultural understanding. Developers respect open source software. Operators value Software as a Service. Both know the virtues of immutable, dynamic data.

The Endatabas SQL dialect draws inspiration from SQL:2011, SQL:2016, and SQL:2023 specifications, making it a sequel all its own. Temporal queries are effortless. Dynamic, semi-structured data is native to Endatabas. The storage and execution engines embrace the decomposition storage model.

We hope you will enjoy building with Endatabas as much as you enjoyed writing your first program.

We will sell Endatabas as a service and release everything as open source.
