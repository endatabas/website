# website

endatabas.com

## Setup

```
sudo apt install bibtex2html m4 python
```

## Build

Only builds the bibliography for now. Requires `bibtex2html` and `m4`.

```sh
./bin/build.sh
```

## Run

Test on a local webserver. Requires `python3`.

```sh
./bin/run.sh
```
