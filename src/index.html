Dynamic SQL in the Clouds.

Endatabas is an immutable database. It profits from today's confluence of scale-out computing power, limitless disk space, and machine learning.

Contemporary to this watershed moment in technology is one of cultural understanding. Developers respect open source software. Operators value Software as a Service. Both know the virtues of immutable data.

Dynamic, semi-structured data is native to Endatabas. Temporal queries are effortless. The storage and execution engines embrace the decomposition storage model.

We hope you will enjoy building with Endatabas as much as you enjoyed writing your first program.

We will sell Endatabas as a service and release everything as open source.
